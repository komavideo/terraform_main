Terraform Project Template
===========================

我的 Terraform 我做主。

## 使用方法

```bash
$ git clone https://github.com/komavideo/terraform_main
$ cd terraform_main
# 目录初始化
$ terraform init
# 检验 tf 文件
$ terraform validate
# 实施计划, 准备资源
$ terraform plan
# 应用部署
$ terraform apply
$ terraform apply -auto-approve
# 摧毁系统
$ terraform destroy
$ terraform destroy -auto-approve
```
